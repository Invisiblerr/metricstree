Flexibility

Characteristics that allow the incorporation of changes in a design. The ability of a design to be adapted to provide functionally related capabilities.

Flexibility = 0.25 * Encapsulation - 0.25 * Coupling + 0.5 * Composition + 0.5 * Polymorphism

For information on the values used to calculate the above expression, see the comments on the node QMOOD Quality Attributes Set in the metrics tree.

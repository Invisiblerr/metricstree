Reusability

Reflects the presence of object-oriented design characteristics that allow a design to be reapplied to a new problem without significant effort.

Reusability = -0.25 * Coupling + 0.25 * Cohesion + 0.5 * Messaging + 0.5 * Design Size

For information on the values used to calculate the above expression, see the comments on the node QMOOD Quality Attributes Set in the metrics tree.

Number Of Overridden Methods (NOOM)

This metric counts the number of attributes of given class.


Included in the set of metrics proposed by M. Lorenz and J. Kidd, See
M. Lorenz, J. Kidd, “Object Oriented Software Metrics”, Prentice Hall, NJ, (1994).


According to
Steffen Herbold, Jens Grabowski, and Stephan Waack. Calculation and Optimization of Thresholds for Sets of Software Metrics Technical Report No. IFI-TB-2010-01, ISSN 1611-1044
the threshold value of this metric is 3.

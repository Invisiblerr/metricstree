Number Of Operations (NOO)

This metric counts the number of operations of given class.

Included in the set of metrics proposed by M. Lorenz and J. Kidd, see

M. Lorenz, J. Kidd, “Object Oriented Software Metrics”, Prentice Hall, NJ, 1994.

According to

L. Rosenberg. Applying and interpreting object oriented metrics. Software Technology Conference, 1998.

the threshold value of this metric is 20.

Bieman-Kang Metrics Set

The metric "Tight Class Cohesion" proposed by Bieman & Kang in [1] is implemented here.

[1] J. M. Bieman and B. Kang, Cohesion and reuse in an object-oriented system, Proceedings of the 1995 Symposium on Software reusability, Seattle, Washington, United States, pp. 259-262, 1995.

Lorenz-Kidd Metrics Set

Lorenz and Kidd’s metrics were defined to measure the static characteristics of software design, such as the usage of inheritance, the amount of responsibilities in a class, etc. The following subset of metrics proposed by Lorenz and Kidd in 1994 [1] is implemented here:

1. Number Of Attributes

2. Number Of Operations

3. Number Of Overridden Methods

4. Number Of Added Methods

[1] M. Lorenz, J. Kidd, “Object Oriented Software Metrics”, Prentice Hall, NJ, 1994.

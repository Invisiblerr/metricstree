Functionality

The responsibilities assigned to the classes of a design, which are made available by the classes through their public interfaces.

Functionality = 0.12 * Cohesion + 0.22 * Polymorphism + 0.22 Messaging + 0.22 * Design Size + 0.22 * Hierarchies

For information on the values used to calculate the above expression, see the comments on the node QMOOD Quality Attributes Set in the metrics tree.

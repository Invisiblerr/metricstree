Extendibility

Refers to the presence and usage of properties in an existing design that allow for the incorporation of new requirements in the design.

Extendibility = 0.5 * Abstraction - 0.5 * Coupling + 0.5 * Inheritance + 0.5 * Polymorphism

For information on the values used to calculate the above expression, see the comments on the node QMOOD Quality Attributes Set in the metrics tree.

Lanza-Marinescu Metrics Set

The following set of metrics proposed by Lanza & Marinescu in [1] is implemented here:

1. Access To Foreign Data

2. Number Of Public Attributes

3. Number Of Accessor Methods

4. Weight Of A Class

[1] M. Lanza, R. Marinescu. Object-Oriented Metrics in Practice. Using Software Metrics to Characterize, Evaluate, and Improve the Design of Object-Oriented Systems. Springer-Verlag Berlin Heidelberg, 2006.

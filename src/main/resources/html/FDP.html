Foreign Data Providers (FDP)

Foreign Data Providers (FDP) is the total number of external classes from which foreign attributes are accessed. Here, these external classes are provisioning data. Ideally, all of the data should be locally available within the class to promote cohesion.

Proposed by M. Lanza, R. Marinescu in order to detect bad smells in code, see

M. Lanza, R. Marinescu. Object-Oriented Metrics in Practice. Using Software Metrics to Characterize, Evaluate, and Improve the Design of Object-Oriented Systems. Springer-Verlag Berlin Heidelberg, 2006.

Desirable value of this metric is less than 3.

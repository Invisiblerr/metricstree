Effectiveness

This refers to a design's ability to achieve the desired functionality and behavior using object-oriented design concepts and techniques.

Effectiveness = 0.2 * Abstraction + 0.2 * Encapsulation + 0.2 * Composition + 0.2 * Inheritance + 0.2 * Polymorphism

For information on the values used to calculate the above expression, see the comments on the node QMOOD Quality Attributes Set in the metrics tree.
